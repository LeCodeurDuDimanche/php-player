# PHP Player
## A package to asynchronously play music using PHP

*This package is in early development stage*

## Requirements

This library relies on ffmpeg, youtube-dl and aplay.
PHP version >= 7.2

## Installation

```
    composer require lecodeurdudimanche\php-player
```
